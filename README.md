### 마일리지 선택제

연세대학교의 수강신청은 다음의 "마일리지 선택제" 방식으로 이루어지고 있습니다. 
수강을 희망하는 정도 만큼 과목별로 마일리지를 배분하고, 마일리지가 동점인 경우, 특정 우선순위에 의하여 수강신청이 이루어집니다.  

#### :key: 특정 우선순위
- (복수) 전공의 일치 여부
- 신청 과목 수
- 졸업 신청 여부
- 총 이수학점 / 졸업이수학점
- 직전 학기 이수학점 / 학기당 수강 가능 학점
